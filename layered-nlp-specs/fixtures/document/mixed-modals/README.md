# Mixed Modals (Document)

Description: Fixtures that test multiple modal types appearing together across paragraphs or sentences.

References:
- https://en.wikipedia.org/wiki/Deontic_modality
- https://en.wikipedia.org/wiki/Modal_verb

Examples:
- "The Tenant may enter..." and "The Tenant shall provide..." in the same document.

Edge cases:
- Same verb used under different modals in adjacent sentences.
- Permissions followed by prohibitions on the same action.
- Mixed modal verbs in the same paragraph.

Difficulty:
- Keeping modal classification stable when signals are close together.
