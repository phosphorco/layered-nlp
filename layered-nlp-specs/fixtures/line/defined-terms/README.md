# Defined Terms

Description: Named labels for entities or concepts (term of art = special contractual label), often written as "Term" means ....

References:
- https://en.wikipedia.org/wiki/Definition
- https://en.wikipedia.org/wiki/Term_of_art

Examples:
- "\"Company\" means ABC Corp."
- "ABC Corp (the \"Company\")"

Edge cases:
- Quoted text that is not a definition.

Difficulty:
- Quote handling, multiword terms, and avoiding false positives in ordinary quoted strings.
